## What is it?

Anthropic's desktop app for collaborative knowledge work — projects, memory, artifacts and plugins. Enterprise-relevant because the model backend can run via Bedrock, Vertex, Azure AI Foundry or an LLM gateway inside the client's own perimeter.

## Why this ring?

We advise on DSGVO-compliant deployment paths (three posts), but have no production rollout referenced yet. The backend recommendation follows the client's existing cloud: AWS → Bedrock, Microsoft → Foundry, Google → Vertex, hard governance → Gateway.

## When to use it — and when not?

- **Use** in evaluations where knowledge workers need an agentic AI workspace and the data must stay in the client's cloud account.
- **Don't** position it where a custom application or an existing Copilot rollout already covers the use case.
