## What is it?

Python framework for data/AI apps — a working UI in hours, not sprints.

## Why this ring?

Our default for pilots and demos: the fastest way to put a RAG or agent prototype in front of client stakeholders. Pilot success decides the real build (→ Next.js).

## When to use it — and when not?

- **Use** for pilots, internal tools and stakeholder demos.
- **Don't** ship it as the production frontend for customer-facing applications.
