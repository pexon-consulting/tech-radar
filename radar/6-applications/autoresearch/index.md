## What is it?

Automated research workflow using LLM agents for literature and topic research. Reference implementation for multi-step reasoning agent patterns.

## Why this ring?

We use it as a pattern reference for building research-oriented agents — the agentic loop pattern adapts well to document analysis or competitive intelligence use cases.

## When to use it — and when not?

- **Use** as a design reference when building multi-step research agents.
- **Don't** deploy it as-is for clients — it's a reference, not a product.
