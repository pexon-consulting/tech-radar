## What is it?

React framework for production web apps — our frontend default for custom AI applications (chat UIs, document workbenches, internal tools).

## Why this ring?

Streaming-friendly (great fit for token streaming), huge ecosystem, easy hiring — the safe default for anything user-facing we build.

## When to use it — and when not?

- **Use** for every custom AI frontend beyond a pilot.
- **Don't** use it where a Streamlit pilot still does the job — don't gold-plate experiments.
