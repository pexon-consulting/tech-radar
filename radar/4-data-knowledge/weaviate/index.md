## What is it?

Open-source vector database with strong hybrid search (BM25 + vectors) and a built-in module ecosystem for vectorization.

## Why this ring?

Our second vector DB next to Qdrant — particularly when hybrid search quality matters out of the box.

## When to use it — and when not?

- **Use** when hybrid retrieval is central and the client likes the managed offering.
- **Don't** run Qdrant *and* Weaviate in the same platform — pick one.
