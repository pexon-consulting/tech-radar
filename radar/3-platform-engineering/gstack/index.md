## What is it?

Opinionated Claude Code setup with 23 tools configured for AI engineering workflows. Useful reference for standardizing local AI dev environments across the team.

## Why this ring?

A reference, not a standard yet — we evaluate it as a starting point for onboarding engineers into Claude Code-based workflows.

## When to use it — and when not?

- **Use** as inspiration when setting up a project's Claude Code tooling; adapt the tool selection to project context.
- **Don't** adopt it wholesale without reviewing each configured tool.
