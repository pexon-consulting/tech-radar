## What is it?

Anthropic's agentic coding assistant — terminal/IDE-based, with hooks, skills and MCP integration for automation beyond code completion.

## Why this ring?

We use it, write about it, and sell a 5-day workshop around it — that is Trial evidence at minimum; once it's the documented default dev setup, promote to Adopt.

## When to use it — and when not?

- **Use** for AI-assisted engineering and for automating internal workflows (hooks, skills, MCP servers).
- **Don't** roll it out without guardrails on client codebases — tool permissions and review discipline first.
